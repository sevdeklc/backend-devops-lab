# Backend DevOps Lab

A practical learning repository for mastering Java Backend Development and DevOps from scratch.

This repository documents my journey of learning and building a complete backend application using modern Java and DevOps technologies.

The final goal is to build, containerize, deploy and monitor a production-style Spring Boot application.
